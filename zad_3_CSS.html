Definicja CSS

HTML:

-strukturalny jezyk znacznikow do budowy trony internetowej

-definiuje zawartosc i jej semantyke, takie jak naglowki, akapity, listy, itp.

-ograniczone mozliwosci formatowania

CSS:

-kaskadowe arkusze stylow pozwalaja na precyzyjne modyfikowanie wygladu elementow HTML

-rozdziela zawartosc od prezentacji

-umozliwia szeroki zakres stylizacji, takich jak kolor, czcionka, marginesy, padding, tlo, itp.

CSS ma nowe mozliwosci w porownaniu z HTML, takie jak:

-wyliczanie i numerowanie list

-ustawienie wlasnych symboli lub numerow dla punktow lub cyfr w listach

-dostosowywanie odstepow miedzy elemenami listy

-kontrola nad wygladem etykiet punktow lub cyfr
