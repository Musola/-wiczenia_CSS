Budowa selektora

  1. Wymienić oraz opisz dwie części slektora

    2. Każda reguła odnosi się do konkretnego elementu (znacznika) i składa się z dwóch części: Element docelowy: Jest to element HTML, na którym chcemy zastosować styl. Może to być np. < div>, < p>, < h1>, itp.
    Klasa, id lub inny atrybut: Jest to dodatkowy atrybut, który identyfikuje konkretny element lub grupę elementów. Może to być klasa (zaczyna się od kropki .nazwa-klasy), id (zaczyna się od hasha #id-elementu) lub inny atrybut (np. [data-atrybut="wartość"]).

  2. Znaczenie nawisów klamrowym {...}

    3. Nawiasy klamrowe otaczają deklaracje właściwości i wartości CSS dla danego selektora. Wewnątrz nawiasów klamrowych definiowane są style, które mają być zastosowane do wybranych elementów.

  3. Opisz cechy

    4. Cechy (właściwości) określają aspekty wyglądu i zachowania elementu HTML, takie jak kolor, rozmiar, marginesy, wypełnienie, itp.

  4. Opisz wartości

    5. Wartości określają konkretne ustawienia dla danej cechy. Na przykład dla cechy color wartością może być nazwa koloru (red, blue), wartość szesnastkowa (#ff0000, #0000ff), czy wartość RGB (rgb(255, 0, 0), rgb(0, 0, 255)).

  5. Kiedy stosujemy znak średnika (;), kiedy nie jest konieczny

    6. Znak średnika ; jest stosowany po każdej deklaracji właściwości i wartości CSS wewnątrz nawiasów klamrowych {}. Jest on konieczny, aby oddzielić poszczególne deklaracje od siebie, jednak nie jest konieczny po ostatniej deklaracji w bloku.
